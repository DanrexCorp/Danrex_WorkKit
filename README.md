# Danrex WorkKit

Первый комбайн **Danrex Tools** — софт-направления **Danrex Corp**. 15+ инструментов в одном приложении.

---

## Возможности

### 🧠 Продуктивность
- ⏱️ Pomodoro-таймер с круговым прогресс-баром
- 📝 Быстрые заметки по Ctrl+Shift+N поверх всех окон
- 📋 История буфера обмена (до 100 элементов, поиск, закрепление)
- 🔔 Напоминания с повтором и откладыванием
- 📸 Скриншот области со встроенным редактором

### ⚙️ Система
- 🧹 Очистка временных файлов, кэша браузеров, логов и корзины
- ⚡ Управление автозагрузкой (вкл/выкл/добавить)
- 📊 Монитор CPU, RAM, диска и GPU в реальном времени
- 📋 Список установленных программ с быстрым удалением
- 🔍 Поиск больших файлов и папок с круговой диаграммой
- 🗑️ Безвозвратное удаление (3 метода: нули, случайные данные, DoD)
- 🛡️ Восстановление ассоциаций файлов
- 🌙 Переключение тёмной/светлой темы Windows в один клик
- 🖥️ Закрепление окон поверх всех (Win+P)
- 📐 Сетка окон (2×2, 3 колонки, свои шаблоны)

### 🌐 Сеть
- 🔗 Сокращение ссылок
- 📡 Тест скорости интернета
- 🔍 Whois и DNS
- 📋 Информация об IP (внешний, внутренний, геолокация)
- 🔒 Проверка портов
- 📊 Пинг и трассировка с графиком

---

## Особенности

- Интерфейс в стиле Windows 11 (скругления, тени, анимации)
- Тёмная и светлая темы с переключением без перезапуска
- Не требует установки
- Полностью бесплатно

---

## Как использовать

1. Скачайте **Danrex WorkKit.exe** из раздела Releases
2. Запустите
3. Выберите инструмент в боковом меню
4. Пользуйтесь

## Требования

- Windows 10 / 11
- Ничего устанавливать не нужно
